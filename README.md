# Modern Professional Portfolio

A high-performance, visually stunning portfolio built with **Next.js 15**, **Tailwind CSS 4**, and **Framer Motion**. This project features advanced animations, responsive design, and a comprehensive showcase of skills, projects, and professional experience.

## âœ¨ Features

- **Dynamic Hero Section**: Engaging introduction with smooth entrance animations.
- **Interactive Skills Showcase**: Technical expertise categorized with staggered reveal animations.
- **Education & Experience**: Chronological timelines of academic and professional milestones.
- **Project Gallery**: Curated selection of work with detailed descriptions.
- **Smooth Navigation**: One-page layout with seamless section transitions.
- **Modern UI/UX**: Built with Radix UI primitives for accessibility and polished with Tailwind CSS 4.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
